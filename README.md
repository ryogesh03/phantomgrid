# PhantomGrid

PhantomGrid is an AI-powered cyber deception agent built for the AMD Developer Hackathon.

## Problem
Cyber threats like phishing, suspicious commands, and malicious login attempts are difficult for beginners and small teams to analyze quickly.

## Solution
PhantomGrid analyzes suspicious cyber inputs, explains the risk, and generates safe decoy logs/responses that simulate a cyber deception layer.

## MVP Features
- Threat input analyzer
- AI-based risk explanation
- Decoy response/log generator
- Detection history dashboard

## Tech Stack
- Python
- Streamlit
- LLM / AI agent
- JSON or SQLite
- GitHub
